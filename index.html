Cloudflare

Bezpečnost a výkon webových stránek

Co je Cloudflare?

Cloudflare je společnost, která poskytuje služby pro zrychlení, zabezpečení a zvýšení dostupnosti webových stránek a aplikací. Funguje mezi návštěvníkem a serverem, kde pomáhá filtrovat škodlivý provoz a rychleji doručovat obsah uživatelům.

K čemu Cloudflare slouží?

Cloudflare pomáhá provozovatelům webů chránit jejich stránky před útoky, zrychlovat načítání obsahu a zajišťovat vyšší spolehlivost služeb. Díky rozsáhlé síti serverů po celém světě dokáže návštěvníkům poskytovat data z nejbližšího místa.

Vybrané služby Cloudflare

CDN (Content Delivery Network)

Ukládá kopie obsahu na servery po celém světě a zrychluje načítání webových stránek.

DDoS Protection

Chrání webové stránky a aplikace před útoky, které se snaží zahltit server velkým množstvím požadavků.

Web Application Firewall (WAF)

Filtruje nebezpečný provoz a pomáhá blokovat pokusy o napadení webových aplikací.
